# Archive policy

Heavy files, private working files, raw logs, internal backups, and unapproved raw materials should be stored outside GitHub, preferably in R2 or another archive location.

Only approved public-safe or partner-safe artifacts should be committed here.

Before removing any local heavy file, first prepare an archive plan, upload or copy the file to the approved archive location, verify backup integrity, and only then remove the local or GitHub-unneeded copy.
